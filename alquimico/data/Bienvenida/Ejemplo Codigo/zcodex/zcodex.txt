Puedes hacer que tus textos destaquen utilizando --strong negritas strong-- o dándole un toque de --em cursiva em--. Además, puedes insertar un salto de línea en medio de un párrafo --br para continuar en la siguiente línea sin crear un nuevo bloque.

--h2 Bloques de Código y Tablas

Para mencionar una variable rápida en un texto, usa código en línea como --code const elixir = true; code--. Para funciones completas, utiliza el bloque de código:

--precode
// Función de transmutación principal
function transmutar(materiaPrima) {
    if (materiaPrima === 'Plomo') {
        return 'Oro';
    }
    return materiaPrima;
}
console.log(transmutar('Plomo'));
precode--

Finalmente, puedes estructurar datos usando tablas:
